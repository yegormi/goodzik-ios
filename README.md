## ğŸš€ Getting Started
### Prerequisites
* iOS 17.0+
* Xcode 16+
* CocoaPods or Swift Package Manager

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/sewing-division-ios.git
   cd sewing-division-ios
   ```

2. Install dependencies:
   ```bash
   pod install
   # or

   ```

   ```bash
   open SewingDivision.xcworkspace
   ```
